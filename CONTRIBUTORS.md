## How to get listed here

Contribute to skeval and your name will be added here. See [CONTRIBUTING.md](CONTRIBUTING.md) for how to get started.

Contributions that count:
- Code (bug fixes, new features, performance improvements)
- Documentation (fixing errors, adding examples, improving clarity)
- Bug reports (well-described issues that help us improve)
- Reviews (thoughtful feedback on pull requests)
- Ideas (feature requests that shaped the roadmap)
